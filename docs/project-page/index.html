Rigid augmentation breaks when objects change topology.

Deformable objects inhabit a high-dimensional, physics-constrained state space. A simple pose offset cannot reach realistic configurations, and a rigid trajectory transform cannot follow material motion after deformation.

DeformGen addresses both sides of this problem: it first perturbs an object through localized dynamics and then warps the demonstration trajectory with a continuous deformation field.

01Topology-coherent states

Local physical disturbances are forward-simulated into plausible configurations.

02Geometry-aware actions

Particle displacement fields adapt both grasp pose and manipulation path.

02Method

A closed loop from physical state diversity to usable demonstrations.

Each generated state remains tied to the source task through deformation-aware trajectory transfer and replay validation.

  1. 01

    Disturb

    Apply localized, controlled perturbations after grasp and forward-simulate the soft-body dynamics.

  2. 02

    Stabilize

    Let the object settle into a physically plausible, topology-coherent target state.

  3. 03

    Warp

    Lift particle displacements into a continuous field that transfers end-effector position and orientation.

  4. 04

    Validate

    Replay, render, export LeRobot data, and retain successful trajectories as policy supervision.
